# IoT Smart Home Security System

An IoT-based Smart Home Security and Safety monitoring system built using an ESP microcontroller, MQTT communication protocol, and a React web dashboard.

The system monitors the home environment using multiple sensors and sends real-time data to a web interface where users can monitor the status of their home.

---

## Features

- Real-time home monitoring
- Motion detection
- Fire detection
- Water leak detection
- Window/door monitoring
- Alarm triggering
- MQTT-based communication
- Web dashboard built with React

---

## System Architecture

ESP Device → MQTT Broker → Backend Server → React Dashboard
